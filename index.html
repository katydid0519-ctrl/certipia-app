수료증 생성기

아래 항목을 입력한 뒤 버튼을 누르면 수료증이 자동 생성됩니다.
비워 둔 항목은 템플릿의 기존 텍스트가 그대로 유지됩니다.

저장된 양식

현재 폼에 입력된 9개 항목을 한 묶음으로 저장합니다. 같은 이름으로 저장하면 덮어씁니다.

대량 생성

아래 영역에 한 줄에 한 명씩 "이름 [탭/공백] 생년월일" 형식으로 붙여넣으세요. 엑셀에서 두 열을 복사해 그대로 붙여 넣어도 됩니다.
• 문서번호는 자동으로 1700000000000~1799999999999 사이의 랜덤값이 부여됩니다.
• 교육생명·생년월일·문서번호를 제외한 항목은 위 폼의 입력값(또는 템플릿 기본값)이 적용됩니다.
• 결과는 한 개의 ZIP 파일로 다운로드됩니다.
